# Intuition Trainer

A cross-platform desktop application for strengthening intuition and focus through guided breathing and number-guessing. Built using Python and PyQt5, this app combines mindfulness with cognitive training in a lightweight and distraction-free environment.

## Features

- Clean, minimalist user interface
- Breathing animation for relaxation and mental preparation
- Two gameplay modes:
  - Think of a number
  - Input a number
- Four difficulty levels: Easy, Medium, Hard, Expert
- Affirmation prompts to encourage mindfulness
- Fully cross-platform: works on Windows, macOS, and Linux
- No dependencies on OS-specific modules (e.g. `winsound` has been removed)

## Installation

Make sure you have Python 3.7 or higher installed.

Install dependencies with pip:
pip install PyQt5


Clone the repository and run:
python main.py

## Files
